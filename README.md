# JavaScript Fundamentals SBA
This project is created with Javascript for the purpose of show casing an understanding of the fundamentals in a practical manner. The program is for processing course assignment, and learn submission information, it calculates learner grades as well.

# User Stories
* As a user, I want to put in course, assignment, and submission information so that the program can process it.
* As a user, I want the program to calculate each learner's weighed average so that I can see how they performed.
* As a user, I want assignments that aren't due to be ignored so they dont mess with grades.
* As a user, I want late submissions to receive a 10% penalty, so that the grading is accurate.
* As a user, I want the program to check that the assignment belongs in the right course, so that it's organized.
